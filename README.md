# Stack Overflow Developer Survey Analysis

## Overview
This project is an exploratory data analysis (EDA) of the Stack Overflow Developer Survey dataset.  
The goal is to understand developer demographics, technologies, salaries, and working patterns using real-world survey data.

---

## Tools
- Python  
- Pandas  
- NumPy  
- Jupyter Notebook (Google Colab)

---

## Dataset
The analysis is based on the official Stack Overflow Developer Survey dataset, which includes responses from thousands of developers worldwide.

Files used:
- survey_results_public.csv
- survey_results_schema.csv

---

## Project Tasks
- Loaded and explored large CSV datasets using Pandas  
- Inspected dataset structure and data types  
- Handled missing values and cleaned data where necessary  
- Performed descriptive statistics (mean, median, mode, percentiles)  
- Filtered data using categorical, numerical, and text conditions  
- Applied grouping and aggregation functions  
- Analyzed distributions of key variables  
- Worked with subsets of data using logical conditions  
- Identified top and most frequent values in key categories  
- Structured results in clear tabular form  

---

## Key Results
- Built a structured workflow for analyzing large survey datasets  
- Identified key trends in developer demographics and technologies  
- Gained practical experience in data cleaning and exploratory analysis  
- Strengthened skills in Pandas for real-world datasets  
- Produced a reproducible Jupyter Notebook analysis
